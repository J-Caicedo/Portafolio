Mi Portafolio

En esta sección encontrarás los proyectos en los que he trabajado, también encontrarás algunas pruebas e inventos. Todo lo que sé hacer con las herramientas que uso.

Para ver en detalle la información de los trabajos que he realizado y en los que he participado puedes dar click en el logo respectivo.

TAXADMINISTRADOR

Es una plataforma web enfocada en la generación de tarjetas de control para los taxistas afiliados a la empresa TELETAXI S.A.S.. Esta plataforma cuenta con otras funcionalidades que permiten a la empresa generar reportes, observar en tiempo real vencimientos de documentación, gestionar la información de los vehículos vinculados a la empresa, notificar sobre vencimientos de documentos importantes y otras funcionalidades más que permiten tener una mayor efectividad en la operación y la gestión de la empresa.

Desarrollador Javascript Full-Stack

BUSSER

Busser es un sistema informático de movilidad urbana que permite a los usuarios del servicio de transporte público observar los buses, rutas y conductores en tiempo real a través de una aplicación móvil. Dicha aplicación muestra al usuario información básica de la empresa y le permite tener conocimiento sobre qué ruta es la óptima para movilizarse de un punto a otro. Busser se alimenta de un API basada en JSONAPI y a través de WebSockets se establece el flujo de la información para la comunicación en tiempo real.

Desarrollador Back-end

YIPGO

YipGO es una aplicación móvil enfocada al proyecto Última Milla de la empresa Metrosinú que permite a los operadores recibir en tiempo real servicios que son solicitados por los usuarios que requieran ser movilizados hacia el paradero más cercano. YipGO se alimenta de un API y hace uso de WebSockets para mayor efectividad en la recepción de servicios.

Desarrollador Back-end

BUSSER DASHBOARD

Es una aplicación web que permite manejar la información principal de las aplicaciones móviles Busser y YipGO tales como rutas, calificaciones, dispositivos, etc. Permite al administrador crear nuevas rutas, paraderos, trayectos y observar métricas que ayudan a mejorar la operación.

Desarrollador Front-end

VRM

VRM es una aplicación web que se encarga de recolectar reportes, manejar sus estados y realizar métricas que buscan mejorar la calidad de los procesos. Ésta aplicación fue realizada para el área de mantenimiento de la empresa Metrosinú.

Desarrollador Javascript Full-Stack

MI PIE

Mi Pie es una plataforma web para la admisión y control de pacientes que padecen diabetes, además permite calcular el grupo de riesgo en que el paciente se encuentra y a partir de esa información programar las próximas citas. En este proyecto se realizó un API basada en JSONAPI para crear los endpoints que atienden a la aplicación web construida con EmberJS.

Desarrollador Javascript Full-Stack

MEDICALL24

Medicall24 es un sistema enfocado a la telemedicina que busca por medio de una aplicación móvil la descongestión de las salas de urgencias en las clinicas y hospitales permitiendo al paciente realizar su consulta con un médico desde la comodidad de su casa. Medicall24 se alimenta de un API y a través de WebSockets se ejecutan ciertas funciones que requieren ejecutarse en tiempo real. Este sistema está integrado con Twilio para la realización de videollamadas.

Desarrollador Back-end

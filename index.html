¡Hola! Bienvenido a mi portafolio.

Mi nombre es John, soy desarrollador web y vivo en la ciudad de Montería. Mi lenguaje de programación favorito es javascript, con él he aprendido muchas cosas y con él desarrollo actualmente. Mi objetivo es usarlo como herramienta para crear soluciones tecnológicas que puedan contribuir al desarrollo de la sociedad.
Me gusta compartir lo que sé con otras personas y que esas personas compartan esa misma filosofía.

Hago parte de una gran comunidad llamada Colombia.dev y soy organizador de la comunidad MonteríaJS, así que eres bienvenid@!

Actualmente uso Github y Bitbucket para guardar y trabajar los proyectos que desarrollo a través del sistema de control de versiones GIT.

Taxadministrador es un sistema de gestión de flotas (taxis) el cual cuenta con varias funcionalidades que ayudan a la empresa a generar tarjetas de control a partir de la información del conductor y del vehículo, notificar sobre vencimientos de documentos importantes, generar reportes, entre otras funcionalidades.

Desarrollador Javascript Full-Stack

Busser es una aplicación android enfocada a la movilidad a través del servicio de transporte público que permite a los usuarios observar buses en tiempo real, información básica, saber qué ruta es la óptima para movilizarse de un punto a otro, etc. En éste proyecto se desarrolló un API en JSONAPI y se crearon namespaces para el envío de información en tiempo real a través de WebSockets.

Desarrollador Back-end

YipGO es una aplicación android enfocada al proyecto Última Milla de la empresa Metrosinú que permite a los operadores del servicio recibir solicitudes en tiempo real de los usuarios que requieran ser movilizados hacia el paradero más cercano.

Desarrollador Back-end

Dashboard Busser es una aplicación web que permite manejar la información principal de las aplicaciones móviles Busser y YipGO. Permite al usuario crear nuevas rutas, paraderos, trayectos y observar métricas que ayudan a observar mejor la información.

Desarrollador Front-end

VRM es una aplicación web responsiva que se encarga de recolectar reportes y realizar métricas a partir de sus atributos. Ésta aplicación fue realizada para el área de mantenimiento de la empresa Metrosinú.
